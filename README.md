## 概要
美少女ゲーム向けランチャーツール

![動作画面](https://pbs.twimg.com/media/FIPA2duaAAIn8hI?format=png&name=small "murrelet")

## 使い方
* ゲームの追加  
ショートカットアイコンをドラッグ＆ドロップ  
または，えろすけidから追加

* 評価の付け方  
評価したいゲームの行を右クリック  
その後，編集から☆をクリック

## 動作環境
Windows 10  

.NET Framework 4.7.2
